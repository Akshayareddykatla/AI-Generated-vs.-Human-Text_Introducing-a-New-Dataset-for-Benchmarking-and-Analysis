# AI-Generated vs. Human Text: Introducing a New Dataset for Benchmarking and Analysis 🤖🧠

This project focuses on detecting and analyzing the differences between AI-generated text and human-written content using Machine Learning and Natural Language Processing (NLP) techniques. The project introduces a structured dataset designed for benchmarking text classification models and improving AI text detection research.

The system performs preprocessing, feature extraction, model training, and evaluation to classify whether a text sample is AI-generated or human-written.
